### table

表格标题：`caption`

表头：`thead`

主体部分：`tbody`

尾部：`tfoot`

表头：`th`

单元格：`td`


```

 <table>
 	<caption>几种页面实现的比较</caption>
    	<thead>
        	<tr><th>实现方法</th><th>代码量</th><th>搜索引擎友好</th></tr>
     	</thead>
 	<tbody>
    	<tr><td>table布局</td><td>多</td><td>差</td></tr>
    </tbody>
 </table>

bootstrap table 中有如下css

.success、.error、.warning、.info

```
